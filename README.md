# MangaXpress

A modern manga reading and purchasing website built using PHP and MySQL.

## Features

- Authentication
- Shopping Cart
- Manga Library
- Search
- User Profile
- Admin Dashboard

## Technologies

- PHP
- MySQL
- JavaScript
- Bootstrap 5
- CSS

## Installation

1. Clone repository

2. Import database.sql

3. Update database connection

4. Start Apache & MySQL

5. Open localhost/MangaXpress
